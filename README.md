# TodoList — консольное приложение

Простое консольное приложение для управления списком задач, написанное на Python.

## Описание

Приложение позволяет вести список задач прямо в терминале: добавлять новые задачи,
просматривать их, отмечать выполненными и удалять. Все задачи автоматически
сохраняются в файл `tasks.json` и загружаются оттуда при следующем запуске.

## Реализованные функции

- Добавление задачи через ввод в консоли
- Отображение всех задач списком
- Отметка задачи как выполненной
- Удаление задачи по номеру
- Меню с пронумерованными опциями
- **Сохранение задач в файл (JSON)**
- **Загрузка задач из файла при старте**
- **Сортировка задач по статусу выполнения**

## Требования

- Python 3.7 или выше

## Инструкция по запуску

1. Склонируйте репозиторий:
   ```
   git clone https://github.com/your-name/your-repo.git
   cd your-repo
   ```
2. Запустите приложение:
   ```
   python3 main.py
   ```
   (на Windows может понадобиться команда `python main.py`)
3. Управляйте задачами через меню, вводя номер нужного пункта.

## Пример работы

```
=== TodoList ===
1 — добавить задачу
2 — показать все задачи
3 — показать задачи (сортировка по статусу)
4 — отметить задачу выполненной
5 — удалить задачу
0 — выход
Выберите пункт меню: 1
Введите текст задачи: Купить хлеб
Задача "Купить хлеб" добавлена.

Выберите пункт меню: 2

--- Список задач ---
1. [Х] Купить хлеб
--------------------
```
